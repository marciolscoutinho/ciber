# ⚖️ Legal Disclaimer — API Security Tester

## Important — Read Before Use

The tool contained in this repository (`api_security_tester.py`) is designed exclusively for **authorized security testing, defensive research, and educational purposes**.

It performs lightweight API security checks mapped to selected categories of the OWASP API Security Top 10 2023. It must not be used against systems without permission.

---

## Authorized Use

This tool may **only** be used against:

- ✅ Systems and networks you personally own
- ✅ Systems for which you have explicit written authorization from the legal owner
- ✅ Intentionally vulnerable applications such as DVWA, WebGoat, Hack The Box, TryHackMe, VAmPI, and local hack-me labs
- ✅ Bug bounty programs, only within the program's defined scope
- ✅ Controlled lab environments with no production data
- ✅ Internal development or staging APIs approved for testing

---

## Prohibited Use

This tool must **not** be used for:

- ❌ Unauthorized access to any system, network, API, or service
- ❌ Scanning or probing systems without explicit written permission

- ❌ Service disruption, denial of service, or system damage
- ❌ Collecting, extracting, or viewing data you are not authorized to access
- ❌ Malicious, criminal, or harmful activity
